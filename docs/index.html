Accordion

Accordion depends on the following sub-components:

  • __trigger: as <input type="checkbox">
  • __header: as <header> with title
  • __body: for content body
  • __content: that encapsulates the content
Example 
<div class="accordion">
  <input type="checkbox" class="accordion__trigger">
  <header class="accordion__header">
    GraphQL
  </header>

  <div class="accordion__body">
    <div class="accordion__content">
      <p>
        Bora pra action. Encare problemas como oportunidades.
        Quebre padrões e atinja a alta performance em todas as áreas da sua vida.
        Ninguém lembra do médio, foque no excelente.
      </p>
    </div>
  </div>
</div>

<div class="accordion">
  <input type="checkbox" class="accordion__trigger" checked>
  <header class="accordion__header">
    React
  </header>

  <div class="accordion__body">
    <div class="accordion__content">
      <p>
        Bora pra action. Encare problemas como oportunidades.
        Quebre padrões e atinja a alta performance em todas as áreas da sua vida.
        Ninguém lembra do médio, foque no excelente.
      </p>
    </div>
  </div>
</div>

Alert

Alerts can be used with <div> and <span> tags to encapsulate alerts.

Types of alerts:

  • Default: Commonly not used.
  • Info: Alert for infos
  • Careful: Alert for careful infos
  • Fluid: Alert with 100% of width
Example 
<div class="alert">Default Alert</div>
<div class="alert alert--info">Info Alert</div>
<div class="alert alert--careful">Careful Alert</div>

Button

Buttons can be used with <a>, and <button> elements.

Types of buttons:

  • Default: Standard button
  • Primary: Provides extra visual red background-color
  • Disabled: Disabled button (with prop or --disabled mod
Example 
<button class="button">Default</button>
<button class="button button--static">Static</button>
<button class="button button--primary">Primary</button>
<button class="button" disabled>Disabled</button>

Cards

Card is a simple component.

You can use the following modifiers:

  • --dashed: adds border-style: dashed.
  • --fluid: adds width: 100%.
  • --ghost: adds lightweight description to card.
  • --spaceless: less padding.

And you have the following subs:

  • __actions: block to add card actions
Example 
<div class="card">
  Pra chegar ao next level você precisa de comprometimento e muito hardwork.
  Não adianta ter conhecimento se você não tem action. Se é pra entrar no jogo,
  vai all-in.

  <div class="card__actions">
    <button class="button button--primary">
      Send
    </button>
  </div>
</div>

<div class="card card--dashed">
Pra chegar ao next level você precisa de comprometimento e muito hardwork.
Não adianta ter conhecimento se você não tem action. Se é pra entrar no jogo,
vai all-in.
</div>

<div class="card card--ghost">
  A definição dos termos técnicos aparecerá aqui. Adicione um texto no campo acima para traduzir!
</div>

Footer

Footer is a simple component with it's sub-components.

You can use the following sub-components:

  • __container: it's an additional class for container.
  • __block: separate blocks of menu lists.
  • __title: list title.
  • __list: menu list.
  • __link: link style for <a>
  • __copyright: component that includes text with copyright infos

Form

Form is a group of components, including __field, __label, __input sub-components.

The __field has the --fluid modifier that adds width: 100%.

The __input has the --high modifier that adds height: 200px.

Example 
<div class="form__field">
  <label for="name" class="form__label">Nome de usuário</label>
  <input type="text" id="name" placeholder="ex. João da Silva" class="form__input space__16">
</div>

<div class="form__field space__20">
  <label for="textao" class="form__label">Conteúdo</label>
  <textarea id="textao" placeholder="Escreva ou cole (⌘ + V) seu texto aqui." class="form__input space__16"></textarea>
</div>

Page Banner

  • --advertising: background color is orange
